# Gestor de Tareas 📝

Un gestor de tareas (To-Do List) moderno y responsivo construido con **HTML, CSS y JavaScript**. Esta aplicación permite organizar actividades diarias mediante un sistema visual de columnas (Kanban) para mejorar la productividad.

## 🚀 Características

- **Diseño Responsivo:** Optimizado para dispositivos móviles, tablets y escritorio mediante Media Queries avanzados.
- **Sistema Kanban:** Visualización de tareas en tres estados: Pendientes, En Progreso y Completadas.
- **Interfaz Intuitiva:** Formulario centralizado para agregar tareas con descripción, prioridad y fecha límite.
- **Contadores Dinámicos:** Seguimiento en tiempo real de la cantidad de tareas en cada columna.
- **Estética Moderna:** Uso de variables CSS para una paleta de colores coherente y tipografías limpias.

## 🛠️ Tecnologías utilizadas

- **HTML5:** Estructura semántica.
- **CSS3:** Diseño personalizado, Flexbox y Media Queries (separados en estilos base y responsivos).
- **JavaScript (Vanilla):** Lógica para la manipulación del DOM y gestión de tareas.
- **Fuentes:** Segoe UI / System-ui.
